أريد منك تحليلًا تقنيًا وتجاريًا دقيقًا جدًا لآلية عمل مزوّدات الذكاء الاصطناعي داخل بعض أدوات الـ IDE/Agents مثل Kilo Code، وبالذات المزود الذي يتيح استخدام اشتراك ChatGPT Plus/Pro أو Codex بدلًا من استهلاك OpenAI API Tokens المدفوعة بنظام Pay-as-you-go.

المطلوب منك أن تشرح لي بوضوح شديد، وبأسلوب احترافي ومبسط في نفس الوقت، النقاط التالية:

1) ما الفرق الحقيقي بين:
- استخدام OpenAI API Key التقليدي
- واستخدام مزود مثل: OpenAI - ChatGPT Plus/Pro داخل أدوات مثل Kilo Code

2) كيف يتم هذا النوع من الربط تقنيًا؟
أريد فهمًا واضحًا للبنية المتوقعة، مثل:
- هل يتم عبر OAuth / Sign in بحساب ChatGPT؟
- هل يتم عبر Codex Entitlements أو Subscription-backed access؟
- هل الأداة تستخدم Endpoint مختلف عن OpenAI API التجاري؟
- لماذا لا يتم الخصم من رصيد API Tokens عند استخدام هذا النوع من المزودات؟

3) ما العلاقة بين:
- اشتراك ChatGPT Plus/Pro الشهري
- Codex
- حدود الاستخدام مثل 5 ساعات أو أسبوعي
- وعدم وجود تكلفة Tokens منفصلة

4) هل هذا التكامل رسمي ومدعوم، أم مجرد التفاف غير مباشر؟
أريدك أن تميز بوضوح بين:
- التكاملات الرسمية
- والتخمينات أو الاستنتاجات غير المؤكدة

5) اشرح لي ما الذي يحدث على الأرجح “خلف الكواليس” خطوة بخطوة منذ لحظة اختيار Provider وحتى بدء إرسال الطلبات للموديل.

6) وضّح لي لماذا لا تستطيع كل التطبيقات الخارجية الاستفادة من اشتراك ChatGPT Plus/Pro بنفس الطريقة، وما الذي يجب أن يتوفر في التطبيق حتى يدعم هذا النوع من الوصول.

7) أريد مقارنة عملية نهائية في جدول أو نقاط واضحة بين:
- API-based access
- Subscription-based ChatGPT/Codex access

8) لو كانت هناك قيود أو حدود أو حالات يتوقف فيها هذا النوع من الاستخدام، اذكرها بوضوح.

اعتمد في إجابتك على:
- الروابط التالية
- وأي مصادر رسمية إضافية موثوقة تجدها
- والصور المرفقة إن أمكن

تعليمات مهمة جدًا:
- لا تكتفِ بشرح عام.
- أريد تحليلًا عميقًا ومحددًا.
- افصل بين “المعلومة المؤكدة من المصدر” و”الاستنتاج المنطقي”.
- لا تفترض أي شيء بدون توضيح درجة اليقين.
- لو وجدت تضاربًا بين المصادر، اذكره بوضوح.
- اختم الإجابة بخلاصة تنفيذية قصيرة جدًا: ما الذي يحدث فعلًا في سطرين أو ثلاثة.
- استخدم عناوين واضحة.
- استخدم العربية.
- لو أمكن، أضف مخططًا نصيًا بسيطًا يوضح مسار العمل بين:
