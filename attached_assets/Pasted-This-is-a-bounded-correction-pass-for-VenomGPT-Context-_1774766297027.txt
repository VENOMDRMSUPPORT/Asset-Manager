  - Observed

  - Not visible / cannot confirm

  - API entitlement failure
  - full fallback chain failure

For tasks like:
"write some information into a file about the error in the attached screen"

the system should understand that the user wants:

- Prevent the agent from trying unrelated reads before writing the requested file when the user’s intent is already clear.

- Investigate and correct the multi-action emission problem in this screenshot path.
